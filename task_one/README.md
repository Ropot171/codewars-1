### Split Strings
**Описание:**
Реализуйте функцию, которая разбивает строку на пары из двух символов. Если длина строки нечетная, последний символ дополняется нижним подчеркиванием ('_').

**Примеры:**
```python
split_string('abc')  # ['ab', 'c_']
split_string('abcdef')  # ['ab', 'cd', 'ef']
```
